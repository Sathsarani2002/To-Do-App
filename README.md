To-Do List Management System
1. Introduction

The To-Do List Management System is a desktop-based application developed using Python and Tkinter.
It allows users to manage daily tasks efficiently by providing features such as task creation, updating, deletion, and completion tracking.
The system uses JSON files for local data storage, ensuring data persistence.

2. Objectives of the System

     To provide a simple and user-friendly task management application

     To help users organize and prioritize daily activities

     To ensure tasks are saved and loaded automatically

     To improve productivity through effective task tracking

3. Scope of the System

        Username-based login system

        Add, view, update, and delete tasks

        Mark tasks as completed

        Local JSON-based data storage

        Offline desktop application

        Single-user usage

4. Features of the System

  User login with input validation

  Add tasks with:

    Task name

    Priority (High / Low)

    Due date

    Category

    View all tasks in a list

    Update selected tasks

    Delete tasks permanently

    Mark tasks as completed

    Automatic saving and loading of tasks

5. Technologies Used

       Python 3.x

       Tkinter (Graphical User Interface)

       JSON (Local data storage)

6. System Requirements

6.1 Hardware Requirements

        Desktop or laptop computer

6.2 Software Requirements

    Python 3.x

    Any operating system that supports Python
